# 第六章 枚举和模式匹配

本章我们将讨论枚举。枚举允许你通过例举类型可能的变体来定义类型。首先，我们定义并使用一个枚举来演示enum如何表示数据的意义。下一步我们将探索一个特别有用的枚举，叫做Option,它表示一个值可能是某种东西或者什么也不是。然后我们将看看如何使用match进行模式匹配，从而轻松的根据一个enum的不同值去执行不同的代码。最后，我们将介绍if let构造如何成为在代码中处理枚举的另一个方便而简洁的习惯用法。
